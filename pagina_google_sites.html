Ferramenta interna — Departamento de Folha

Validação de Folha de Pagamento automatizada

Confere apontamentos, empréstimos consignados e coparticipação de plano de saúde contra o extrato mensal do sistema Domínio — em segundos.

⬇ Baixar validador.py Ver no GitHub →
Visão geral

O que o script valida

O validador.py lê o PDF do extrato mensal gerado pelo Domínio (a fonte da verdade) e cruza com os três arquivos Excel que o escritório recebe dos clientes. Qualquer diferença vira uma linha no relatório final.

🗂️

Apontamentos

  • Vale refeição local e cartão
  • Vale transporte cartão e desconto
  • Plano de saúde (8111) e odonto (202)
  • Fator moderador (203)
  • Horas extras 50% e 100%
🏦

Empréstimos

  • Cruza contrato por contrato
  • Detecta parcelas com valor errado
  • Detecta contratos no Excel não descontados
  • Detecta contratos na folha ausentes no Excel
🏥

Coparticipação

  • Rubrica 227 — PS informativa por funcionário
  • Rubrica 203 — fator moderador
  • Compara Excel do cliente vs folha processada
Horas Extras: o Excel de apontamentos informa as horas; o script calcula o valor (horas × salário-hora × multiplicador) e compara com a folha. Nenhuma digitação manual necessária.
Configuração inicial

Instalação — uma única vez

1

Instale o Python 3.10 ou superior

Baixe em python.org/downloads e marque "Add Python to PATH" durante a instalação (Windows).

2

Baixe o validador.py

Clique no botão acima ou acesse o GitHub e salve o arquivo validador.py em uma pasta fixa, ex: C:\Huty\validador\

3

Instale as dependências (só na primeira vez)

Abra o Prompt de Comando (cmd) e execute:

pip install pdfplumber pandas openpyxl xlrd
Uso mensal

Como rodar a validação

Coloque todos os arquivos do mês em uma pasta (ex: C:\Folhas\06-2026\) e execute um dos comandos abaixo no Prompt de Comando.

Opção 1 — pasta automática (mais fácil)

# O script detecta os arquivos automaticamente pelo nome python validador.py --pasta C:\Folhas\06-2026

Opção 2 — arquivos individuais

python validador.py \ --extrato Aponatmentos__-_Sistema.pdf \ --apontamentos APONTAMENTOS_06-2026.xls \ --emprestimos EMPRESTIMOS_06-2026.xlsx \ --coop DESCONTOS_DE_COOPARTICIPAC_A_O_06-2026.xls

Resultado

O relatório Validacao_Folha_06-2026.xlsx é criado na mesma pasta, com 4 abas:

  • Resumo — status geral por módulo
  • Apontamentos — divergências de lançamentos (vermelho = erro, amarelo = aviso)
  • Empréstimos — contratos com valor errado ou ausentes
  • Coparticipação — diferenças na rubrica 227 e 203
